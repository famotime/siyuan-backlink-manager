## 当前结论

## 当前范围内的关键缺口

1. `nearby` 在普通块场景缺少“同一所属标题节”边界约束。
2. `nearby` 在列表场景仍会把邻居深层子树整棵带入。
3. `core` 在列表子块场景仍偏大，不利于首屏定位具体引用点。
4. `extended` 仍按“最近任意标题”切段，而不是“所属标题节”。
5. 无标题文档尚未先尝试结构容器兜底。
